# 2.0.1

Release date: 2026-04-27

## Motivation

Align the published artifacts with the renamed repository and update the Android compatibility
floor.

## Changes

- changed both module package repository URLs to `arch-event-observer`
- raised Android `minSdk` from 20 to 23
- replaced the Java-style merge lock in `ResponseLiveData` with idiomatic Kotlin `Any`

## Compatibility notes

- Android API 20-22 is no longer supported.
- No public API behavior changed.
